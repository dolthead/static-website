About StaticPWA

This is a minimal static site used for demo and teaching purposes. It uses Bootstrap from a CDN and a simple header navigation to move between pages.

Purpose

The goal is to have a tiny, easy-to-serve static website that can be extended into a PWA or used as an exercise for routing and service workers.
